Run the examples in Eclipse
----------------------------------------
1. Copy dynamicreports-examples folder to your workspace
2. Copy all files from lib directory located in dynamicreports-xxx-project-with-dependencies file 
   to dynamicreports_examples/lib/
3. Copy all files from dist directory located in dynamicreports-xxx-project-with-dependencies file 
   to dynamicreports_examples/lib/
4. Run eclipse and import project into workspace
   (choose File-Import-Existing projects into workspace, ...)
5. Build the project
