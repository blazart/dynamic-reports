How to run the examples in NetBeans
----------------------------------------
1. Copy /examples/netbeans/dynamicreports-examples from the archive to your NetBeansProjects folder e.g. /NetBeansProjects/dynamicreports-examples
2. Copy all files from /lib and /dist folders from the archive to /NetBeansProjects/dynamicreports_examples/lib/
3. Run NetBeans and open the project
4. Build the project
