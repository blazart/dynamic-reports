DynamicReports

DynamicReports is an open source Java reporting library based on JasperReports. It lets you create sophisticated reports quickly without needing to use a visual designer. DynamicReports is extremely easy to understand.

The DynamicReports provides benefits you will not get elsewhere:

  • dynamic report design
  • inherited reports
  • increase reporting productivity
  • decrease training time
  • easy to understand
  • easy to maintain reports
  • type safety
